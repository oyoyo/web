Some assumptions

Part of the power of DITA is the versatility of the schema. It can be constrained and specialized for your purpose. FontoXML is as flexible as DITA itself and any implementation partner can configure FontoXML using the FontoXML SDK.

For this standard-edition, we work with a few pre-defined settings. That means that we make a few assumption about how you are using DITA.

No mixed content

Mixed content means that withing a certain DITA element, both text, inlines-type elements ánd block-type elements can be placed. To get to a consistent experience, we automatically remove this inconsistency from your content by always wrapping text or inlines in a <p> element.

Refer to DITA 1.3

At the moment, this standard edition of FontoXML is focused on DITA 1.3 only, the latest edition of DITA. The editor will open DITA 1.2 maps and topics but automatically upgrade them to OASIS DITA 1.3.

Notice:

We expect to support 'Save as DITA 1.2' soon.

No specialization

This version of FontoXML does not support specializations. If you want to try FontoXML with your specialized content, please contact us to configure it for your needs.
